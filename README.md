# 🚀 Multi-user Portfolio Platform (Laravel)

A scalable multi-user portfolio management system built باستخدام Laravel، يتيح للمستخدمين إنشاء وإدارة مشاريعهم الشخصية عبر لوحة تحكم متكاملة.

---

## ✨ Features

- 🔐 Authentication System (Login / Register)
- 👥 Multi-user support
- 📊 Admin Dashboard
- 🗂️ Project Management (CRUD)
- 🌐 RESTful API for frontend integration
- 🧾 Structured JSON responses

---

## 🛠️ Tech Stack

- Laravel
- PHP
- MySQL
- REST API
- Postman (for testing)

## ⚙️ Installation

```bash
git clone https://github.com/Yazan-AbdAlazeez/Portfolio-Api
cd Portfolio-Api
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
